NexusStagingClient

class NexusStagingClient(baseUrl: String = "https://oss.sonatype.org/service/local/", username: String, password: String)

NexusStagingClient will create a staging repository and upload the given files to the given staging repository Creating a staging repository avoid having split artifacts

The process is outlined in this document (from 2014!): https://support.sonatype.com/hc/en-us/articles/213465868-Uploading-to-a-Staging-Repository-via-REST-API

Parameters

baseUrl

: the url of the nexus instance, defaults to the OSSRH url

username

: your Nexus username. For OSSRH, this is your Sonatype jira username

password

: your Nexus password. For OSSRH, this is your Sonatype jira password
